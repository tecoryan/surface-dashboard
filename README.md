## About The Project

Surface Dashboard is a visually sophisticated analytics interface designed to present key business metrics in a clear, engaging format. It showcases how thoughtful design can transform raw data into an intuitive and beautiful user experience.

This project serves as a practical implementation of the design principles taught in Bubble's Design Fundamentals course, covering everything from typography and color theory to responsive layout and visual hierarchy.

## Key Features

- **Key Metric Cards:** Display total revenue, sales, subscriptions, and churn with trend indicators
- **Data Visualization:** Clean presentation of numerical data with percentage changes
- **Transaction Feed:** Real-time list of recent transactions with user and amount details
- **User Activity Feed:** Latest signed-up users displayed with timestamps
- **Modern Aesthetic:** Polished interface with careful attention to spacing, typography, and color

## Built With

- **[Bubble.io](https://bubble.io/)** - The core no-code platform for visual development
- **Bubble's Design Tools** - For implementing typography, color schemes, and responsive layouts
- **Bubble's Repeating Groups** - For displaying dynamic transaction and user feeds
- **Custom Styling** - Carefully crafted visual elements including cards, badges, and data displays

## Design Principles Applied

Building this dashboard through the "Design Fundamentals" course provided deep insights into visual design for web applications. Key design principles learned and applied include:

- **Visual Hierarchy:** Using size, color, and spacing to guide attention to the most important metrics
- **Typography:** Selecting and scaling fonts for readability and aesthetic appeal
- **Color Theory:** Creating a cohesive color palette that enhances data comprehension
- **Card-Based Design:** Organizing information into digestible, visually distinct containers
- **Whitespace:** Using spacing effectively to reduce cognitive load and improve scanability
- **Consistency:** Maintaining uniform styling across all interface elements
- **Data Presentation:** Formatting numbers and percentages for immediate understanding
- **Responsive Considerations:** Designing layouts that adapt gracefully across screen sizes
